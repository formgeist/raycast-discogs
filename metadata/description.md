Browse and search your Discogs record collection directly from Raycast — no browser required.

Your entire collection is fetched and cached locally, so search is instant. Filter by format, sort by recently added, and open any release to see its full tracklist, label, and pressing details. Artwork is shown throughout.

**Features**

- Fuzzy search across title, artist, label, genre, and style
- Filter by format: Vinyl, CD, Cassette, Digital, and more
- Sort by recently added, title, artist, or year
- Full release detail view with tracklist and cover art
- Opens directly to the release page on Discogs
- Collection cached locally for fast repeat opens, with background revalidation
- Authenticated via a personal access token — no OAuth required
